# 🐝 Beecrowd Solutions - C#

Este repositório contém as minhas resoluções para os problemas da plataforma [Beecrowd](https://www.beecrowd.com.br/). As soluções são focadas em lógica de programação e algoritmos utilizando **C#**.

## 🚀 Como os problemas são resolvidos?

Para garantir que o código seja aceito pelo juiz online, todos os exercícios seguem este padrão:
- Utilização de `CultureInfo.InvariantCulture` para evitar erros com pontos/vírgulas decimais.
- Leitura de dados via `Console.ReadLine().Split(' ')` para entradas na mesma linha.
- Formatação de saída com `:F2`, `:F4` conforme exigido pelo enunciado.
