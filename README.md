# Beecrowd Solutions - C#

Este repositorio contem as minhas resolucoes para os problemas da plataforma [Beecrowd](https://www.beecrowd.com.br/). As solucoes sao focadas em logica de programacao e algoritmos utilizando **C#**.

## Como os problemas sao resolvidos?

Para garantir que o codigo seja aceito pelo juiz online, todos os exercicios seguem este padrao:
- Utilizacao de `CultureInfo.InvariantCulture` para evitar erros com pontos/virgulas decimais.
- Leitura de dados via `Console.ReadLine().Split(' ')` para entradas na mesma linha.
- Formatacao de saida com `:F2`, `:F4` conforme exigido pelo enunciado.
